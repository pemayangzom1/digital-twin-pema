## Project Overview
This project aims to build an AI Digital Twin using Retrieval-Augmented Generation (RAG). The system will simulate professional interviews based on personal career data.

## Objectives
- Build an AI interview assistant
- Store professional data in vector database
- Generate evidence-based responses
- Demonstrate AI engineering workflow

## Technical Requirements
- Vector Database: Upstash Vector
- LLM: Claude / Groq
- MCP server integration
- GitHub collaboration workflow

## Functional Requirements
- Accept job descriptions
- Retrieve relevant profile data
- Generate interview responses
- Provide hiring recommendation

## Non-Functional Requirements
- Reliable retrieval
- Professional tone responses
- Secure data handling

## Acceptance Criteria
- AI can conduct interview autonomously
- Responses grounded in personal data
- Final recommendation generated
